## 🎨 自分好みの色で、毎日を彩る家計簿

「家計簿はつけなきゃいけないけど、地味でなかなか続かない……」  
そんな悩みを解決するために、**Wallet Keeper** は生まれました。

その日の気分や自分のスタイルに合わせて、  
パステルカラーやビビッドカラー、シックなモノトーンまで、UI全体を自由自在にカスタマイズ。

お気に入りのデザインに囲まれることで、  
億劫になりがちな収支管理を「ちょっと楽しい時間」に変えてくれます。

---

## ✨ 特徴

### 1. 🎨 豊富なカスタムテーマ
15種類以上のカラープリセットから、ボタン一つでデザインを切り替え可能。

**おすすめテーマ：「大人の落ち着き」**  
深い紺色を基調とした、落ち着きのあるダークモードテーマです。  
夜の家計簿タイムも、目に優しくリラックスして作業できます。

---

### 2. ⚡ 直感的なスピード入力
「まずは簡単に入力できること」を最優先に設計。

複雑な手順を排除し、1列に並んだフォームから迷わず収支を登録できます。  
現在の技術を最大限に活かし、今の自分たちにできる「使いやすさ」を形にしました。

---

### 3. 📊 視覚的な収支分析
Chart.jsを使用した円グラフと棒グラフで、お金の流れを一目で把握。

- **カテゴリ別支出割合（円グラフ）**  
  → どのカテゴリに使いすぎているか即座に判別

- **月別収支推移（棒グラフ）**  
  → 1年間の収入・支出のバランスを比較

※グラフの文字色は、選択したテーマに合わせて自動で最適化されます。
